## What It Does

This app demonstrates seamless Bitcoin Lightning wallet integration for web applications:
- **Connect** any WebLN-compatible wallet (Alby, Coinos, Flash Wallet, etc.)
- **Instant payments** without copy-pasting invoices
- **One-click experience** for Bitcoin transactions on the web
